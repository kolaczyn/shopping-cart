# Shopping Cart API

## How to setup

- setup Golang
- setup MongoDb
- setup Postgres
- seed database:
  - 1. comment out `db.SeedDb()` in `main.go`
  - 2. run the app with `go run main.go`
  - 3. comment the line from step 1.

Once those steps are completed, you can run the app:

```sh
go run main.go
```
